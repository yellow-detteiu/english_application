# english_application
DMM生成AI Camp 23-5

### 英会話の学習に使えるアプリです

　以下の3つの機能があります
1. 日常英会話
　- AIとの音声会話や、文法の添削を受けることができます。
2. シャドーイング
　- AIが読み上げた英文を真似て、それを評価してもらう機能です。
3. ディクテーション
　- AIが読み上げた英文を入力し、それを評価してもらう機能です。

### アプリ起動手順
- 「pip install -r requirements.txt」で必要なライブラリをインストール
- 「streamlit run main.py」でアプリを起動
